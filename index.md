# XP Bot 사용 설명서

이 봇은 텔레그램 그룹에서 유저들의 활동을 기록하고,  
경험치(XP)와 랭킹, 이벤트 추첨을 간단하게 진행할 수 있도록 도와주는 봇입니다.

---

## 1. 기본 개념

- **XP(경험치)**  
  그룹 내에서 대화, 참여 등 활동에 따라 자동으로 누적되는 포인트입니다.

- **레벨**  
  일정 XP가 쌓이면 자동으로 레벨이 올라갑니다.

- **랭킹**  
  같은 기간 동안 누가 가장 많이 참여했는지 확인할 수 있습니다.

---

## 2. 일반 유저 명령어

아래 명령어는 그룹 채팅에서 직접 사용할 수 있습니다.

- `/my_xp`  
  현재 내 XP와 레벨을 보여줍니다.

- `/rank`  
  그룹 내 상위 랭킹 일부를 보여줍니다.  
  (예: TOP 10 또는 상위 사용자 리스트)

- `/event_rank`  
  관리자가 설정한 이벤트 기간 동안의 XP 랭킹을 보여줍니다.

---

## 3. 관리자 전용 명령어

관리자로 등록된 계정만 사용할 수 있는 명령어입니다.  
(관리자 등록 방식은 별도 안내에 따릅니다.)

- `/add_xp @사용자 50`  
  특정 유저에게 XP를 수동으로 추가합니다.

- `/remove_xp @사용자 50`  
  특정 유저의 XP를 수동으로 차감합니다.

- `/set_time 3 8`  
  새벽 3시부터 8시까지 봇이 활동을 기록하지 않도록 설정합니다.  
  - 입력 형식: `/set_time 시작시각 끝시각`  
  - 시각은 0~23 사이의 숫자 (24시간 기준)

- `/lottery`  
  현재까지의 XP를 기준으로 추첨 이벤트를 실행합니다.  
  추첨 결과는 그룹 채팅에 공지됩니다.

---

## 4. 사용 흐름 예시

1. 봇을 그룹에 초대합니다.
2. 관리자가 기본 설정을 합니다.
   - 필요하면 `/set_time`으로 기록 중지 시간대 설정
3. 유저들이 평소처럼 대화하고 참여하면 XP가 자동으로 쌓입니다.
4. 이벤트 기간에는 `/event_rank`로 랭킹을 확인합니다.
5. 마무리로 `/lottery` 명령어로 추첨을 진행합니다.
