# Terminal.Fi XP Bot 설명서

커뮤니티에서 활동하면 XP를 얻고 레벨이 올라가는 텔레그램 봇입니다.  
아래는 **일반 유저 명령어**와 **관리자 명령어**만 정리한 문서입니다.  
(OWNER 전용 명령어는 설명하지 않습니다.)

---

## 📌 일반 명령어

그룹 안에서 누구나 사용할 수 있는 명령어입니다.

### `/stats`
내 현재 **XP, 레벨, 초대 수** 등 전체 스탯을 확인합니다.

### `/ranking`
경험치 기준 **TOP 10 랭킹**을 보여줍니다.

### `/daily`
하루 한 번, **일일보상 XP**를 수령합니다.

### `/mylink`
나만의 **Terminal.Fi 초대 링크**를 생성합니다.

### `/myinvites`
지금까지 내가 **초대한 인원 수**를 확인합니다.

### `/invites_ranking`
초대 인원 기준 **초대 랭킹 TOP**을 조회합니다.

### `/join`
진행 중인 **추첨 이벤트**에 참가합니다.

### `/event_rank`
관리자가 설정한 **캠페인 기간 동안의 XP 랭킹**을 확인합니다.

---

## 🔧 관리자 명령어

관리자 전용 명령어이며, **봇과 DM으로 사용하는 것을 권장**합니다.  
(OWNER 전용 명령어는 이 문서에서 제외)

### 1) 기본 조회

- `/chat_id <@handle 또는 user_id>`  
  → 특정 유저의 Telegram `user_id` 조회

- `/list_admins`  
  → 등록된 관리자 목록 확인

- `/ref_user <@handle>`  
  → 특정 유저가 초대한 인원 수 확인

- `/user_stats <@handle>`  
  → 특정 유저의 XP·레벨·초대·활동 등 전체 스탯 조회

---

### 2) 활동 요약

- `/today`  
  → 오늘 기준 메인 그룹 활동 요약 (KST)

- `/week`  
  → 최근 7일간 활동 요약

- `/range YYYY-MM-DD YYYY-MM-DD`  
  → 지정한 기간의 활동 데이터 요약

---

### 3) XP 키워드 관리

- `/add_xp_bonus <word> <xp>`  
  → 메시지에 해당 **키워드가 포함되면 보너스 XP 지급**

- `/add_xp_block <word>`  
  → 해당 키워드가 포함된 메시지에는 **XP 미지급(차단)**

- `/del_xp_word <word>`  
  → 등록된 XP 키워드(보너스/차단) 삭제

- `/list_xp_words`  
  → 현재 등록된 XP 관련 키워드 목록 확인

---

### 4) XP 정책 설정

- `/set_cooldown <초>`  
  → 같은 유저가 다시 XP를 받을 때까지의 **최소 간격(쿨다운)** 설정

- `/set_daily_cap <XP>`  
  → 유저 1명이 하루에 얻을 수 있는 **최대 XP**를 제한

- `/set_inv_xp <XP>`  
  → **초대 1명당 지급할 XP** 설정

---

### 5) 캠페인 설정

- `/set_campaign <YYYY-MM-DD> <YYYY-MM-DD>`  
  → 캠페인 XP 집계 기간 설정 (`/event_rank` 결과에 반영)

- `/clear_campaign`  
  → 설정된 캠페인 기간 및 관련 데이터 초기화

---

### 6) 봇 운영 시간 & 상태

- `/set_time <시작> <종료>`  
  → KST 기준, **XP 기록이 활성/비활성 되는 시간대** 설정 (0~23시 정수)

- `/bot_off`  
  → 봇의 XP 기록 및 주요 기능 **일시 중지**

- `/bot_on`  
  → 중지된 봇 기능 **다시 활성화**

- `/bot_status`  
  → 현재 봇 상태 및 시간 설정 등 확인

---

### 7) 수동 XP & 추첨

- `/add_xp <@handle> <XP>`  
  → 특정 유저에게 **XP 수동 지급**

- `/sub_xp <@handle> <XP>`  
  → 특정 유저의 **XP 차감**

- `/lottery [분] [당첨자수]`  
  → 그룹 추첨 시작  
  예: `/lottery 5 3` → **5분 뒤 3명 추첨**

- `/lottery_end <인원수>`  
  → 진행 중인 추첨을 종료하고, 지정한 인원 수만큼 당첨자 선정
