Terminal.Fi XP Bot 설명서

커뮤니티 활동을 XP로 기록하고, 랭킹·초대·캠페인 이벤트를 운영할 수 있는 텔레그램 봇입니다.

XP 랭킹 초대 리워드 캠페인 기간 집계 관리자용 제어 명령어

📌 일반 명령어

그룹 내 모든 유저 사용 가능
  • /stats
    내 현재 XP, 레벨, 초대 수 등 전체 스탯을 확인합니다.
  • /ranking
    경험치 기준 TOP 10 랭킹을 보여줍니다.
  • /daily
    하루 한 번, 일일보상 XP를 수령합니다.
  • /mylink
    나만의 Terminal.Fi 초대 링크를 생성합니다.
  • /myinvites
    지금까지 내가 초대한 인원 수를 확인합니다.
  • /invites_ranking
    초대 인원 기준 TOP 랭킹을 조회합니다.
  • /join
    진행 중인 추첨 이벤트에 참가합니다.
  • /event_rank
    설정된 캠페인 기간 동안의 XP 랭킹을 확인합니다.

🔧 관리자 명령어

관리자 전용 · DM 사용 권장
기본 조회
  • /chat_id <@handle 또는 user_id>
    특정 유저의 Telegram user_id를 조회합니다.
  • /list_admins
    등록된 관리자 목록을 확인합니다.
  • /ref_user <@handle>
    특정 유저가 초대한 인원 수를 조회합니다.
  • /user_stats <@handle>
    특정 유저의 XP, 레벨, 초대 수 등 전체 스탯을 확인합니다.
활동 요약
  • /today
    오늘 기준 메인 그룹 활동 요약을 보여줍니다. (KST)
  • /week
    최근 7일간의 활동 요약을 보여줍니다.
  • /range YYYY-MM-DD YYYY-MM-DD
    지정한 기간의 활동 데이터를 요약해서 보여줍니다.
XP 키워드 관리
  • /add_xp_bonus <word> <xp>
    메시지에 해당 단어가 포함되면 추가 XP를 지급하도록 설정합니다.
  • /add_xp_block <word>
    해당 단어가 포함된 메시지에는 XP를 지급하지 않도록 차단합니다.
  • /del_xp_word <word>
    등록된 XP 키워드(보너스/차단)를 삭제합니다.
  • /list_xp_words
    현재 등록된 XP 관련 키워드 목록을 확인합니다.
XP 정책 설정
  • /set_cooldown <초>
    한 유저가 XP를 다시 받을 때까지 최소 간격(쿨다운)을 설정합니다.
  • /set_daily_cap <XP>
    유저 1명이 하루에 얻을 수 있는 최대 XP를 설정합니다.
  • /set_inv_xp <XP>
    초대 1명당 지급할 XP 수치를 설정합니다.
캠페인 설정
  • /set_campaign <YYYY-MM-DD> <YYYY-MM-DD>
    캠페인 XP 집계 기간을 설정합니다. /event_rank에 반영됩니다.
  • /clear_campaign
    설정된 캠페인 기간 및 관련 데이터를 초기화합니다.
봇 운영 & 상태
  • /set_time <시작> <종료>
    KST 기준으로 XP 기록이 활성/비활성 되는 시간을 설정합니다 (0~23시 형식).
  • /bot_off
    봇의 XP 기록 및 주요 기능을 일시 중지합니다.
  • /bot_on
    중지된 봇 기능을 다시 활성화합니다.
  • /bot_status
    현재 봇의 상태 및 시간 설정 등을 확인합니다.
수동 XP & 추첨
  • /add_xp <@handle> <XP>
    특정 유저에게 XP를 수동으로 지급합니다.
  • /sub_xp <@handle> <XP>
    특정 유저의 XP를 차감합니다.
  • /lottery [분] [당첨자수]
    그룹에서 추첨을 시작합니다. 예: /lottery 5 3 → 5분 뒤 3명 추첨.
  • /lottery_end <인원수>
    진행 중인 추첨을 종료하고 지정 인원 수만큼 당첨자를 뽑습니다.
