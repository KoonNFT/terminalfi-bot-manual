Commands that any user can run inside the group.

  • /stats Show your XP, level, invites and basic stats.
  • /ranking Display the Top 10 XP ranking.
  • /daily Claim your daily XP reward (once per day).
  • /mylink Generate your personal Terminal.Fi invite link.

Admin-only commands. It is recommended to run most of these via DM with the bot. (OWNER-only commands are intentionally omitted.)

Basic Lookup
  • /chat_id <@handle or user_id> Get the Telegram user_id of a specific user.
  • /list_admins List all registered admins.
  • /ref_user <@handle> Check how many users a specific user has invited.
  • /user_stats <@handle> See full stats (XP, level, invites, etc.) for a user.
Activity Summary

XP Keyword Controls
  • /add_xp_bonus <word> <xp> Give bonus XP when a message contains this keyword.
  • /add_xp_block <word> Block XP gain when a message contains this keyword.
  • /del_xp_word <word> Remove a registered bonus/block keyword.
  • /list_xp_words List all XP-related keywords.
XP Policy
  • /set_cooldown <seconds> Set the minimum interval between XP gains per user.
  • /set_daily_cap <XP> Set the maximum daily XP per user.
  • /set_inv_xp <XP> Set XP amount awarded per invite.
Campaign Controls
  • /set_campaign <YYYY-MM-DD> <YYYY-MM-DD> Set the campaign period for XP (used by /event_rank).
  • /clear_campaign Clear campaign period and related data.
